# The Untold Life

A simple, interactive web project showcasing basic web development skills. This project utilizes HTML, CSS, and JavaScript to create a dynamic user experience.

## Features

*   Interactive elements
*   Responsive design (basic)
*   Clean user interface

## Tech Stack

*   HTML
*   CSS
*   JavaScript

## Live Demo

[the-untold-life.vercel.app](https://the-untold-life.vercel.app)

## Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/Shiva-sainiiii/The-Untold-Life.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd The-Untold-Life
    ```
3.  Open `index.html` in your web browser.
